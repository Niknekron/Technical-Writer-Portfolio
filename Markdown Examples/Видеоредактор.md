
== '''Описание ситуации''' ==
В платформе SmartPlayer существует функциональность, которая позволяет редактировать текущий или новый видеоконтент. Данная страница пояснит как пользоваться видеоредактором и какие возможности заложены в это решение.
== '''Требования к контенту''' ==
Для корректной работы видеоконтента необходимо соблюдать следующие требования:<br>
Разрешение - до UHD 3840X2160 (если платформа поддерживает данное разрешение);<br>
Количество кадров: 25-30;<br>
Битрейт: 8000;<br>
Кодек: H 264;<br>
Формат видео - mp 4.
{{Note|Для устройств под управление ОС Tizen, WebOS и BrightSign рекомендуется убирать аудио-часть из видеоконтента.|warn}}
== '''Алгоритм действия''' ==
Первым шагом для пользователя является авторизация / регистрация на платформе SmartPlayer.
Далее пользователь должен найти раздел «Контент» и выбрать, с помощью клика, категорию «Видеоредактор».
[[File:Видеоредактор контента.png|thumb|center| Пример отображения категории «Видеоредактор» |800px]]
После клика по категории пользователю откроется главная страница, в котором присутствуют четыре основных элемента работы с видеоконтентом:
# Зона контента — позволяет добавить видеоконтент из личного кабинета или с устройства;
# Панель настроек — позволяет совершить действия с обработанным контентом;
# Тулбар с контентом — отображает информацию об имеющемся или добавленном для редактирования контенте;
# Настройки конвертации видеоконтента - '''по умолчанию - не отображается'''. Открывается щелчком мыши по загруженному видеоконтенту и позволяют работать и взаимодействовать с контентом.
[[File:4_элемента.png|thumb|center| Пример отображения элементов работы с видеоконтентом |800px]]
Подробнее о каждом элементе будет описано ниже.
=== '''Зона контента''' ===
Зона контента в исходном состоянии представляет из себя окно с описанием, которое предлагает добавить видеоконтент.
[[File:Зона_видеоконтента.png|thumb|center| Пример отображения зоны для редактирования видеоконтента |800px]]
Видеоконтент в зону можно добавить двумя способами:
* Перетащить с помощью drag-n-drop из локального хранилища.
* Добавить из тулбара с контентом (П.3.) имеющийся видеоконтент.
=== '''Панель настроек''' ===
В панель настроек входят четыре элемента:
* Анимация — предоставляет настройки анимации для начала и конца видеоконтента;
* Настройки — позволяет выбрать название и место хранения в личном кабинете для видеоконтента;
* Сохранить — позволяет пользователю сохранить обработанный контент;
* Выйти — возвращает пользователя в раздел «Контент» личного кабинета.
[[File:Панель_настроек.png|thumb|center| Пример отображения элементов в панели настроек|800px]]
=== '''Тулбар с контентом''' ===
Тулбар с контентом представляет из себя разделенный на две части:
# Задачи — зона, в которой располагается добавленный к редактированию видеоконтент;
# Контент — зона, в которой расположен доступный в личном кабинете видеоконтент.
[[File:Тулбар_с_частями.png|thumb|center| Пример отображения тулбара с контентом|800px]]
{{Note|Контент в задачах можно выделить с помощью клика и удалить по специальной иконке в виде мусорной корзины.
[[File:Иконки_деействия.png|thumb|center| Пример отображения иконок «Выбрать» и «Удалить»|800px]]|warn}}
=== '''Настройки конвертации видеоконтента''' ===
{{Note|Данный блок настроек изначально скрыт. Чтобы активировать настройки видеоредактора необходимо кликнуть на видеоконтент загруженный в зону контента. |warn}}
Открыв настройки для конвертации видеоконтента пользователь сможет изменить следующие параметры:
* Продолжительность (время начала / время окончания) — время, которое указывается вручную и позволяет обрезать видеоконтент;
* Переключатель «Обрезать продолжительность видео до целых значений» — активирует / деактивирует обрезку длины видео до целых значений. Округляет в меньшую сторону;
* Размеры контента (ширина / высота) — позволяет вручную указать размеры видеоконтента по ширине и высоте;
* Чекбокс (галочка) «Сохранить пропорции» — включить / выключить параметр сохранения пропорций по ширине и высоте;
* Битрейт — прописывается значение битрейта. Измеряется в (Кбит/сек);
* Доступный битрейт — позволяет выбрать из списка доступный битрейт под определенный разрешения: '''360p, 480p, 720p, 1080p, 1440p, 2160p ''';
* Количество кадров в секунду — указывается количество проигрываемых кадров в секунду;
* Поворот контента — поворачивает контент по часовой стрелке на определенное количество градусов с шагом 90;
* Чекбокс (галочка) «Удалить звук из видео» — удаляет из видеоконтента аудиодорожку;
* Чекбокс (галочка) «Использовать пресет» — позволяет выбрать пресет для Android устройства из списка;
* Выбор пресетов — список доступных пресетов для устройств с: настройкой вручную, оптимизацией под Andorid 4.0, с оптимизацией под Android 4.4.
[[File:Настройки_параметров.png|thumb|center| Пример отображения параметров настройки видеоредактора |800px]]
=== '''Пресеты для Android-устройств''' ===
Чтобы настроить работу пресетов для Android - устройств необходимо отредактировать следующие поля:
* Видеокодек — позволяет выбрать видеокодек для кодирования видеоряда. Доступен — libx264;
* Аудиокодек — позволяет выбрать аудиокодек для кодирования аудиодорожки. Доступен — aac;
* Профиль кодека — позволяет выбрать один из трёх профилей для кодирования: baseline (базовый, для слабых устройств), main (основной, с балансом между качеством и совместимостью), high (высокий, ориентирован на максимальное качество);
* Уровень кодека — регулирует и оптимально комбинирует разрешение, частоту кадров, битрейт и сложность декодирования;
* Пресет — позволяет выбрать доступный пресен из списка;
* Качество видео при кодировании — позволяет выбрать значение качества выбора при кодировании. Работает для кодирования ffmpeg.
[[File:Пресеты_для_редактора.png|thumb|center| Пример отображения настроек пресетов для Andorid OS |800px]]
== '''Обработка видео''' ==
{{Note|Для запуска процесса редактирования видео пользователю необходимо нажать на кнопку «Обработать» внизу экрана. Если пользователь хочет изменить внесенные изменения, то ему необходимо нажать на кнопку «Отменить».
[[File:Кнопки_обработать_и_отменить.png|thumb|center| Пример отображения кнопок «Обработать» и «Отменить»|800px]]|warn}}
В итоге запуститься процесс обработки и конвертации видеоконтента.
== '''Завершение работы''' ==
После завершения работы с видеоредактором пользователю предложат придумать название для обработанного файла и место его хранения в личном кабинете, по аналогии с кнопкой «Настроить» и панели настроек.
