== '''Описание ситуации''' ==
Данный виджет выводит на устройства информацию с расписанием занятий, при этом позволит виджету работать с необходимым фоновым контентом в рамках функционала личного кабинета SmartPlayer.
== '''Серверная логика''' ==
Серверная часть отсутствует. Данный виджет все данные для вывода получает через Excel-файл. После получения информации, обрабатывает данные и выводит их в трансляцию через функционал личного кабинета SmartPlayer.
== '''Добавление виджета в личный кабинет''' ==
Чтобы добавить данный виджет в личный кабинет, пользователю необходимо запросить у специалиста команды технической поддержки архив с содержимым. Он будет предоставлен в формате .zip-архива.
[[File:Пример_виджета.png|thumb|center| Пример виджета в формате .zip-архива|800px]]
После этого, пользователь должен авторизоваться в личном кабинете и перейти в раздел «Контент».
[[File:Раздел_контент_личного_кабиента.png|thumb|center| Пример отображения раздела «Контент» в личном кабинете |800px]]
В разделе «Контент» необходимо перенести данный виджет из локального хранилища в личный кабинет. После того как пользователь перенесет виджет в личный кабинет, он отобразится в списке с контентом.
[[File:Отображение_виджета расписания.png|thumb|center| Пример отображения переноса виджета в личном кабинете SmartPlayer|800px]]
== '''Главная страница виджета''' ==
После переноса виджета для первичной настройки пользователю необходимо дважды кликнуть по иконке виджета. Данное действие перенесет его на главную страницу.
[[File:Виджет_стартовая_страница.png|thumb|center| Пример отображения главной страницы виджета|800px]]
На главной странице виджета пользователь может кликнуть по иконке в виде шестеренки, чтобы перейти в настройки виджета.
[[File:Настройка_виджета_занятий.png|thumb|center| Пример отображения иконки для перехода в настройки виджета|800px]]
== '''Кнопка «Настройки»''' ==
В разделе «Настройки» пользователь может настроить следующие параметры:
* цвет текста — позволяет с помощью палитры настроить цвет текста в виджете;
* первый цвет логотипа — позволяет с помощью палитры настроить часть цвета логотипа;
* второй цвет логотипа — позволяет с помощью палитры настроить оставшуюся часть логотипа;
* загрузить пользовательский SVG логотип — открывает модальное окно загрузки файла из локального хранилища. Загруженный файл должен быть в формате SVG;
* название учебного заведения — позволяет прописать текстом название учебного заведения;
* загрузить файл Excel — открывает модальное окно для загрузки Excel-файла с расписанием занятий.
[[File:Настройкки_виджета_занятий.png|thumb|center| Пример отображения настроек виджета|800px]]
=== '''Загрузка Excel-файла с расписанием''' ===
Подготавливаемый Excel-файл с расписанием должен содержать следующие поля:
* дата;
* номер занятия (в течение дня);
* время;
* номер аудитории.

== '''Завершение настройки виджета''' ==
После завершения работ по настройке виджета пользователь должен нажать на кнопку «Применить настройки» в правой нижней части экрана.
[[File:Применение_настрок.png|thumb|center| Пример отображения кнопки «Применить настройки»|800px]]
Далее виджет растянется на весь экран и пользователь должен нажать на кнопку «Сохранить», внизу по центру экрана.
[[File:Сохран.png|thumb|center| Пример отображения кнопки «Сохранить»|800px]]
== '''Загрузка и скачивание настроек''' ==
В процессе взаимодействия с настройками виджета пользователь может загрузить или выгрузить настройки виджета. Каждое из указанных действий можно сделать с помощью отдельных иконок в интерфейсе виджета. Иконки расположены в виджете снизу по центру.
[[File:Иконки_загрузки.png|thumb|center| Пример отображения иконок в настройках виджета |800px]]
== '''Добавление тега виджету''' ==
Чтобы упростить использование виджета в трансляции и не создавать большое количество трансляций рекомендуется выставить тег для подготовленного виджета. Для этого в личном кабинете необходимо кликом выделить настроенный виджет и нажать на иконку «Редактировать» (отображается в виде карандаша).
[[File:Редактирование_виджета.png|thumb|center| Пример отображения иконки «Редактировать» в личном кабинете SmartPlayer|800px]]
В открывшейся странице пользователю необходимо найти строку «Тег» и прописать тег которым пользователь хочет обозначить виджет. После добавления тега нажать на кнопку «Сохранить» в нижней части экрана.
{{Note|Чтобы тег считывала система его необходимо ввести в строке и нажать «Enter».|warn}}
[[File:Указание_тега.png|thumb|center| Пример отображения тега виджета|800px]]
Далее пользователю необходимо перейти в раздел «Устройства» и выбрать нужное устройство кликом.
[[File:Выбор_устройства_тега.png|thumb|center| Пример отображения устройства|800px]]
После выбора устройства в правой части экрана откроется панель управления настройками. Пользователю необходимо нажать на строку «Редактировать».
[[File:Устройство_для_редактирования.png|thumb|center| Пример отображения панели настроек со строкой «Редактировать»|800px]]
По клику на строку «Редактировать» откроется страница с параметрами устройства. На странице необходимо найти параметр «Тег» и указать тот же тег что был указан для виджета.
{{Note|Чтобы тег считывала система его необходимо ввести в строке и нажать «Enter».|warn}
[[File:Добавление_тега_к_устройства.png|thumb|center| Пример отображения параметра «Тег»|800px]]
После ввода тега нажать на кнопку «Сохранить» в нижней части экрана.
